# Spicebridge

A Next.js + Tailwind CSS website with glass morphism design, animations, and a chat widget. Design inspiration from [Promethean Consulting](https://prometheanconsult.com/).

## Features

- **Hero Carousel** - Rotating hero images from your assets folder
- **Glass Morphism** - Frosted glass effect on background, overlays, buttons, and cards
- **Animations** - Framer Motion animations on buttons, cards, loading states
- **Chat Widget** - "Chat with us" floating button that opens a WhatsApp chat form
- **Mobile Responsive** - Optimized for both mobile and desktop

## Getting Started

**Requirements:** Node.js >= 18.17 (upgrade if needed: [nodejs.org](https://nodejs.org))

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- Next.js 14 (App Router)
- Tailwind CSS
- Framer Motion
- TypeScript

## Project Structure

- `src/app/` - App router pages and layout
- `src/components/` - Reusable components
- `public/assets/` - Images (copied from root `assets/` folder)
